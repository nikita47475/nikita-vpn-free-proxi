# Компания мини-игры VPN

Это сайт-панель и локальный Python-контроллер для VPN-подобного SOCKS-туннеля.

## Картинка Сверху

Положи свою картинку сюда:

```text
\x6d\x69\x6e\x69\x5f\x67\x61\x6d\x65\x73\x5f\x76\x70\x6e\x5f\x73\x69\x74\x65\x2f\x61\x73\x73\x65\x74\x73\x2f\x6c\x6f\x67\x6f\x2e\x70\x6e\x67
```

Страница сама покажет `\x6c\x6f\x67\x6f\x2e\x70\x6e\x67`. Если файла нет, будет показана заготовка `\x61\x73\x73\x65\x74\x73\x2f\x70\x6c\x61\x63\x65\x68\x6f\x6c\x64\x65\x72\x2e\x73\x76\x67`.

## Быстрый Визуальный Режим

Открой `\x69\x6e\x64\x65\x78\x2e\x68\x74\x6d\x6c` двойным кликом. Кнопки будут работать как интерфейс-демо, но настоящий VPN не запустится, потому что обычный HTML-файл не может запускать Python.

## Настоящий Туннель

1. На удаленном сервере запусти `\x74\x75\x6e\x6e\x65\x6c\x5f\x73\x65\x72\x76\x65\x72\x2e\x70\x71`.
2. В `\x76\x70\x6e\x5f\x63\x6f\x6e\x66\x69\x67\x2e\x6a\x73\x6f\x6e` укажи:
   - `\x73\x65\x72\x76\x65\x72\x5f\x6d\x6f\x73\x74` - IP твоего сервера
   - `\x73\x65\x72\x76\x65\x72\x5f\x70\x6f\x72\x74` - порт сервера, обычно `\x38\x34\x34\x33`
   - `\x74\x6f\x6b\x65\x6e` - такой же секретный токен, как на сервере
3. Запусти `\x72\x75\x6e\x5f\x76\x70\x6e\x5f\x70\x61\x6e\x65\x6c\x2e\x62\x61\x74`.
4. Открой:

```text
\x68\x74\x74\x70\x3a\x2f\x2f\x31\x32\x37\x2e\x30\x2e\x30\x2e\x31\x3a\x38\x30\x38\x30
```

Кнопка `Запустить VPN` стартует локальный SOCKS5-прокси на:

```text
\x31\x32\x37\x2e\x30\x2e\x30\x2e\x31\x3a\x31\x30\x38\x30
```

Чтобы браузер реально шел через туннель, поставь в настройках браузера SOCKS5:

```text
Host: \x31\x32\x37\x2e\x30\x2e\x30\x2e\x31
Port: \x31\x30\x38\x30
```

## Важно

Сайт, загруженный в Google или открытый в браузере, не может сам менять IP всей сети. Для защиты всей сети нужен системный VPN-клиент, драйвер TUN/TAP или VPN на роутере. Этот проект дает настоящий TCP/SOCKS-туннель для приложений, которые ты направишь на `\x31\x32\x37\x2e\x30\x2e\x30\x2e\x31\x3a\x31\x30\x38\x30`.

## Команда Для Сервера

На сервере нужен TLS-сертификат. Для теста можно создать самоподписанный сертификат через OpenSSL, затем запустить:

```powershell
\x70\x79\x74\x68\x6f\x6e\x20\x74\x75\x6e\x6e\x65\x6c\x5f\x73\x65\x72\x76\x65\x72\x2e\x70\x79\x20\x2d\x2d\x68\x6f\x73\x74\x20\x30\x2e\x30\x2e\x30\x2e\x30\x20\x2d\x2d\x70\x6f\x72\x74\x20\x38\x34\x34\x33\x20\x60
  \x2d\x2d\x63\x65\x72\x74\x20\x63\x65\x72\x74\x73\x2f\x73\x65\x72\x76\x65\x72\x2e\x63\x72\x74\x20\x60
  \x2d\x2d\x6b\x65\x79\x20\x63\x65\x72\x74\x73\x2f\x73\x65\x72\x76\x65\x72\x2e\x6a\x65\x79\x20\x60
  \x2d\x2d\x74\x6f\x6b\x65\x6e\x20\x22\x63\x68\x61\x6e\x67\x65\x2d\x74\x68\x69\x73\x2d\x6c\x6f\x6e\x67\x2d\x72\x61\x6e\x64\x6f\x6d\x2d\x73\x65\x63\x72\x65\x74\x22
```
